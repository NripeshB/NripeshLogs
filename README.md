# Blog & Portfolio Platform

A full-stack MERN application that combines a personal portfolio with a structured blogging platform.
Visitors can browse blogs and articles publicly, while authenticated users can interact through likes and comments.
Authors can create and manage blogs containing multiple articles, and administrators oversee moderation and user roles.

The project is designed to scale from a single-author setup into a multi-author content platform.

### Tech Stack
- Frontend: React, Vite, Material UI, Redux Toolkit
- Backend: Node.js, Express
- Database: MongoDB (Mongoose)
- Authentication: JWT
- Deployment: Vercel (frontend), Render (backend)
